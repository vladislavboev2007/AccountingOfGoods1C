# 1C: Управление складом и складскими операциями

## 📦 О программе
**1C: Управление складом и складскими операциями** — это прикладное решение на платформе 1С:Предприятие 8.3, разработанное для комплексной автоматизации складского учёта на предприятиях малого и среднего бизнеса.

Программа позволяет вести учёт товаров, контролировать остатки, управлять себестоимостью, проводить инвентаризации, загружать данные из внешних файлов (Excel, XML) и формировать аналитическую отчётность.

🎯 ## Для кого данная программа (версия 3.0)
- **Менеджеры склада** - Оперативное управление складом: приёмка, отгрузка, перемещение, списание товаров, загрузка данных из Excel/XML
- **Руководители склада** - Контроль остатков, анализ движения товаров, утверждение инвентаризаций, управление минимальными остатками
- **Начальники отдела логистики** - Планирование поставок, управление запасами, работа с поставщиками и договорами, аналитика
- **Администраторы** - Настройка системы, управление пользователями и правами доступа, резервное копирование

## 📋 Версии программы

### 🟢 Базовая версия 1.2 (ПП.01)
- **Справочники** (Номенклатура, Контрагенты, Склады, Договоры)
- **Документы** (Приход товара, Расход товара, просмотр движений документа)
- **Учёт остатков**	(Хранение и просмотр текущих остатков на складе)
- **Отчёты** (Товары на складах, Себестомость товаров, Продажи)
- **Партионный учет товаров при отгрузке** (в зависимости от метода расчёта себестоимости (ФИФО/ЛИФО/Среднее))

### 🔵 Расширенная версия 3.0 (ПП.11)

Включает все функции базовой версии плюс:
- **Документ Инвентаризация** - Документ со статусами: Подготовлен → Сверка → Завершён. Автоматическое создание корректирующих документов (приход/расход) по итогам сверки
- **Инвентаризация**
  - **Печатная форма** - Ведомость инвентаризации с пустыми полями для ручного заполнения
  - **Загрузка из скана** - Распознавание отсканированных ведомостей через Tesseract OCR с автоматическим заполнением фактических количеств и примечаний
  - **Статусная модель** - Три статуса: Подготовлен → Сверка → Завершён
  - **Автокорректировка** - Автоматическое создание документов Приход (излишки) и Расход (недостачи)
- **Отчёт "Валовая прибыль"** - 	Отчёт по валовой прибыли с расчётом рентабельности
- **Загрузка данных**
  - **Автоматическое определение настроек** - Система сама определяет подходящую настройку по структуре файла
  - **Множественные настройки** - 	Сохранение нескольких настроек для разных форматов Excel одного контрагента
  - **Загрузка из XML (УПД)** - Загрузка универсальных передаточных документов из XML
  - **Сопоставление номенклатуры** - Автоматическое или ручное сопоставление, создание новых позиций с записью соответствий
  - **Сохранение соответствий** - Автоматическое запоминание сопоставленных позиций в регистре
- **Выгрузка данных XML (УПД)** - Выгрузка документов Расход в стандартном XML-формате для ЭДО
- **Управление ценами**
  - **Цены закупки** - Автоматическая фиксация цен закупки в регистре
  - **Цены продажи** - Автоматическая фиксация цен продажи для контрагентов
  - **Цветовая индикация** - Настройка цветовых порогов отклонения цен (желтый, красный, зеленый)
- **Контроль остатков**
  - **Минимальный остаток** - Установка и контроль минимальных остатков товаров
  - **Предупреждения** - Автоматическое предупреждение при отгрузке ниже минимального остатка
- **Безопасность**
  - **Ролевая модель (RBAC)** - 4 роли: Администратор, Менеджер склада, Руководитель склада, Начальник логистики
  - **Маска ввода** - Маска ввода для КПП (1234 56 789)
  - **Контроль ИНН** - Автоматическая проверка длины ИНН в зависимости от вида контрагента
  - **Журнал регистрации** - Аудит действий пользователей
  - **Шифрование паролей** - Хранение паролей в зашифрованном виде
- **Администрирование**
  - **Резервное копирование** - Выгрузка/загрузка информационной базы в файл .dt
  - **Настройка констант** - Управление методом расчёта себестоимости, порогами цветов, реквизитами организации

## 🚀 Быстрый старт
### Установка
1. Установите платформу 1С:Предприятие 8.3 (учебная или коммерческая версия)
2. Откройте Конфигуратор → «Файл» → «Открыть конфигурацию» 
3. Загрузите файл конфигурации (`.cf` или выгрузка `.dt`)
4. Запустите 1С в режиме «Предприятие»

### Настройка
1. Заполните константы: Название организации, ИНН, КПП 
2. Создайте пользователей: Администратор, Менеджер склада, Руководитель склада, Начальник логистики 
3. Назначьте роли: Настройте права доступа через роли в Конфигураторе 
4. Выберите метод расчёта себестоимости: ФИФО / ЛИФО / Среднее

### Для работы с Tesseract OCR
1. Скачайте и установите [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/releases)
2. При установке отметьте языковые пакеты **Russian** и **Latin/English**
3. Проверьте установку: `tesseract --list-langs` (должны быть `eng` и `rus`)
4. Используйте кнопку «Загрузить из скана» в документе «Инвентаризация»
